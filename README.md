# OrangeHRM Automation Framework

### **Overview**
Industry-standard Selenium + TestNG Hybrid Automation Framework built using Page Object Model for OrangeHRM web application.

### **Tech Stack**
- **Language:** Java 11
- **Build Tool:** Maven
- **Testing Framework:** TestNG
- **Automation Tool:** Selenium WebDriver 4.x
- **Reporting:** Extent Reports
- **Design Pattern:** Page Object Model

### **Key Features**
- **Maven Standard Directory Layout** - `src/main/java` for framework, `src/test/java` for tests
- **Page Object Model** - Separate classes for each page
- **TestNG Listeners** - Automatic screenshot capture on test failure
- **Extent Reports** - Detailed HTML reports with logs and screenshots
- **Data Driven** - Excel integration using Apache POI
- **Config Driven** - Browser, URL, credentials managed via `Config.properties`
